# Overview

DeadlineSyncAI is an intelligent AI automation system that reads Telegram messages, detects deadlines automatically using NLP + AI, and syncs them directly to Google Calendar with reminders.

The system supports:

✅ English  
✅ Bangla  
✅ Banglish  

It can detect:

- Assignments
- Quizzes
- Exams
- Presentations
- Submission Deadlines
- Meetings

# How It Works
1.User sends message in Telegram group
2.Bot reads the message
3.AI extracts date & deadline
4.Event gets added to Google Calendar
5.Reminder notification is automatically scheduled
